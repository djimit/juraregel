# Contributor Covenant Code of Conduct

## Onze Pledge
Wij beloven om iedereen die bijdraagt aan JuraRegel met respect te behandelen, ongeacht achtergrond, ervaring, of rol.

## Onze Standaarden
- Welkom en inclusief gedrag
- Respect voor verschillende viewpoints
- Constructieve feedback
- Focus op wat het beste is voor de community

## Handhaving
Schendingen kunnen worden gemeld via GitHub Issues. Elke klacht wordt beoordeeld en beantwoord.

## Attribution
Dit Code of Conduct is aangepast van Contributor Covenant v2.1.
