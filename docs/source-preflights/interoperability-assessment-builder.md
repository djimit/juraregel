# Interoperability Assessment Builder

## Doel

Maak een machineleesbaar assessment voor diensten die onder EU 2024/903 kunnen vallen.

## Input

- digitale publieke dienst
- grensoverschrijdend effect
- betrokken bestuurslagen
- gebruikte standaarden, APIs en registers

## Bronnen

- EU Interoperable Europe Act 2024/903
- NORA
- Forum Standaardisatie
- API-register
- TOOI/ROO

## Output

- juridische, organisatorische, semantische, technische en governance-impact
- ontbrekende interoperability solutions
- publiceerbaar rapport na human approval

## Niet Doen

Geen rapport namens een organisatie publiceren zonder expliciete goedkeuring.
