# Digitale Dienst Compliance Check

## Doel

Bundel losse compliance-signalen voor een digitale overheidsdienst.

## Input

- URL
- organisatie
- productnaam
- optionele OpenAPI URL

## Bronnen

- UPL
- TOOI/ROO
- DigiToegankelijk dashboard/register
- API-register
- data.overheid.nl CKAN API
- Forum Standaardisatie

## Output

- uniforme productnaam
- toegankelijkheidsstatus
- API-register/lifecycle signaal
- datasetmetadata signaal
- concrete ontbrekende acties

## Niet Doen

Geen publiek organisatierapport publiceren zonder human approval.
