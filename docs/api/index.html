Introductie

De JuraRegel API biedt programmatische toegang tot de volledige Living Compliance Engine. Genereer assessments, voer juridische analyses uit, en monitor compliance — alles via REST endpoints.

Base URL: http://localhost:8080/api/v1/
OpenAPI Spec: http://localhost:8080/openapi.json
Swagger UI: http://localhost:8080/docs

Kenmerken

  • 40+ REST endpoints — Templates, assessments, agents, scoring, policies
  • AI-gestuurde analyse — DPIA Agent, FRIA Agent, Regulatory Monitor
  • Real-time compliance scoring — 7-criteria gewogen model
  • Policy-as-Code — Testbare compliance policies
  • Multi-tenant — Row Level Security per organisatie
  • Audit trail — Immutable logging van alle acties

Snelstart

1. API Starten

# Met Docker Compose (aanbevolen)
docker compose up -d

# Of lokaal met Python
pip install -r requirements.txt
python3 api/run.py

# API beschikbaar op http://localhost:8080

2. Health Check

curl http://localhost:8080/health

# Response:
{"service": "juraregel-api", "status": "healthy", "version": "4.0.0"}

3. Eerste Assessment Genereren

curl -X POST http://localhost:8080/api/v1/assessments/ \
  -H "Content-Type: application/json" \
  -d '{
    "organisation_id": "org-1",
    "assessment_type": "dpia",
    "template_id": "dpia_pre_scan",
    "parameters": {"verwerking": "WOZ-AI Waardering"}
  }'

4. Compliance Score Berekenen

curl -X POST http://localhost:8080/api/v1/compliance/score \
  -H "Content-Type: application/json" \
  -d '{
    "measures_implemented": 5,
    "measures_total": 8,
    "training_current": true
  }'

# Response:
{"score": 72.5, "classification": "good", ...}

Assessments

CRUD + workflow voor compliance assessments (draft → review → approved → published → archived).

POST/api/v1/assessments/
Maak een nieuw assessment aan
GET/api/v1/assessments/
Lijst assessments (filterbaar op type, status)
POST/api/v1/assessments/{id}/submit
Draag in voor review
POST/api/v1/assessments/{id}/approve
Keur het assessment goed (vereist FG rol)
POST/api/v1/assessments/{id}/publish
Publiceer het approved assessment
POST/api/v1/assessments/{id}/archive
Archiveer het published assessment
Workflow: draft → in_review → approved → published → archived
Elke transitie wordt gelogd in de audit trail met actor, timestamp, en reden.

Policy-as-Code

Compliance-regels geëxcodeerd als testbare, automatische policies.

GET/api/v1/policies/
Lijst beschikbare policies
POST/api/v1/policies/evaluate
Evalueer alle policies tegen een context
Voorbeeld 
// Request
{
  "context": {
    "purpose": "Recruitment AI",
    "data_categories": ["Naam", "Geslacht", "Leeftijd"],
    "security_measures": ["encryptie"],
    "ai_systems": true,
    "bias_examined": false
  }
}

// Response
{
  "total_policies": 4,
  "compliant": 1,
  "non_compliant": 3,
  "compliance_rate": 25.0,
  "violations": [
    {
      "policy_id": "ai-act-art10-data-governance",
      "article": "EU AI Act Art. 10(2)(f)",
      "message": "Training data is niet onderzocht op biases",
      "severity": "high",
      "remediation": "Voer bias-auditing uit"
    }
  ]
}
POST/api/v1/policies/evaluate/{policy_id}
Evalueer een specifieke policy
Geïmplementeerde policies:
  • AVG Art. 25 — Data Minimization
  • AVG Art. 32 — Security of Processing
  • EU AI Act Art. 10 — Data Governance
  • EU AI Act Art. 14 — Human Oversight

Integratie Gids

Python Client

import httpx

client = httpx.Client(base_url="http://localhost:8080")

# Genereer DPIA
resp = client.post("/api/v1/templates/dpia_pre_scan/generate", json={
    "organisation": "Mijn Organisatie",
    "parameters": {"verwerking": "Klantanalyse"}
})
dpia = resp.json()["document"]

# Bereken compliance score
resp = client.post("/api/v1/compliance/score", json={
    "measures_implemented": 6,
    "measures_total": 8,
    "training_current": True
})
score = resp.json()["score"]

cURL

# Health check
curl http://localhost:8080/health

# Run DPIA Agent
curl -X POST http://localhost:8080/api/v1/agents/dpia/run \
  -H "Content-Type: application/json" \
  -d '{"organisation_id": "org-1", "processing_activity": {...}}'

JavaScript/TypeScript

const response = await fetch("http://localhost:8080/api/v1/compliance/score", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    measures_implemented: 5,
    measures_total: 8,
    training_current: true
  })
});
const { score, classification } = await response.json();
